# unicodedata

# unicodedata.east_asian_width(char: str) -> str

Returns the East Asian width of a Unicode character. The width is one of the following values:

  • F: Fullwidth
  • H: Halfwidth
  • N: Neutral
  • Na: Narrow
  • W: Wide
  • A: Ambiguous
